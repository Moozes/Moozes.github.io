Short Description

This is the first project to do when learning a new technology because it helps you familiarize yourself with CRUD operations which are present almost in every web application, this app started simple, where it had only add and delete operations, then with time I added other features like edit, filter by todo status (All, active, or completed), and Delete all completed todos, and because user feedback is always important, I added a feedback message component that appears to confirm that the action taken by the user was successful.
